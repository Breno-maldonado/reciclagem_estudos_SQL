# 📚 Estudos de SQL

Este repositório foi criado para armazenar meus estudos, exercícios e anotações sobre **SQL (Structured Query Language)**.

O objetivo é centralizar todo o conteúdo que estou aprendendo, acompanhar minha evolução e servir como material de consulta para projetos futuros.

---

## 🎯 Objetivos

- Aprender os fundamentos da linguagem SQL.
- Praticar consultas e manipulação de dados.
- Entender modelagem de banco de dados relacional.
- Aprender boas práticas na escrita de consultas.
- Criar uma base de conhecimento para futuras consultas.
